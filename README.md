# Jharika Looks ✨ - Premium Clothing Catalog (B2B/B2C)

Jharika Looks ek minimalist aur mobile-first e-commerce catalog hai jo **Sarees** aur **Readymade** garments ke liye banaya gaya hai. Ye platform B2B (Wholesale) aur B2C (Retail) dono ko support karta hai.

## 🚀 Key Features

- **Dynamic Home Page:** Auto-playing banner carousel aur promo banners.
- **Hybrid Shopping:** Retail aur Bulk buying dono ke liye support.
- **MOQ System:** Har product ke liye Minimum Order Quantity (MOQ) set karne ka option.
- **WhatsApp Checkout:** Bina kisi payment gateway ke, direct WhatsApp par order details bhejne ki suvidha.
- **Admin Control Panel:** Website ke banners, logo, category photos, aur products ko live edit karne ke liye built-in CMS.
- **Self-Healing Data:** LocalStorage ka use karke data ko browser mein save rakhta hai.

## 🛠️ Tech Stack

- **Frontend:** React.js (via Babel for single-file support)
- **Styling:** Tailwind CSS (Modern & Responsive)
- **Icons:** Lucide React
- **Animations:** CSS3 Transitions & Keyframes
- **Deployment:** Vercel / GitHub Pages / Local Host

## 💻 How to Run

Aapko kisi software ki zaroorat nahi hai. Bas niche diye gaye steps follow karein:

1. `index.html` file ko download karein.
2. File par double-click karke kisi bhi modern browser (Chrome, Edge, Safari) mein kholein.
3. Website live chalne lagegi!
